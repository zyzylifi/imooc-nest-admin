# Vue3+NestJS 全栈开发企业级管理后台

## 代码结构
```bash
├── src # 项目源码
|  ├── exception # 异常处理
|  ├── modules # 模块源码
|  |  ├── auth # 登录模块
|  |  ├── book # 电子书模块
|  |  ├── menu # 菜单模块
|  |  └── user # 用户模块
|  └── utils # 工具方法
```
